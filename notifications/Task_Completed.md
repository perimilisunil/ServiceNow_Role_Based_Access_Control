# Notification: Task Completed (Optional)

- **Email Name:** Task Completed Notification
- **Trigger:** After Update on Project Task, when `state` changes to Completed
- **Recipients:** Project Manager (task owner)
- **Subject:** `Task ${number} marked Completed by ${assigned_to}`
- **Purpose:** Notifies the PM that a Team Member has finished their assigned task, so it can be reviewed and moved to Closed.
