# User Assignments

Describes group and role assignments for each user.

- **Sunil Kumar:** Project Managers (group) -> Project Manager (role)
- **Sasi:** Team Members (group) -> Team Member (role)
- **Kumar:** Team Members (group) -> Team Member (role)
- **Sasanka:** Team Members (group) -> Team Member (role)
- **Deepak:** Team Members (group) -> Team Member (role)

All Team Members inherit the "Team Member" role privileges via group membership, and Sunil inherits "Project Manager" privileges the same way. This indirect (group-based) assignment pattern is the recommended ServiceNow practice, as it simplifies onboarding/offboarding.
